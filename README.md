# 华中农业大学学士学位论文开题报告书 LaTeX 模板

> 🎯 **华中农业大学学士学位论文开题报告书 LaTeX 模板**

## 📋 简介

这是华中农业大学学士学位论文开题报告书的 LaTeX 模板，完全按照学校官方开题报告书格式制作。

模板包含：
- 封面信息表格（题目、姓名、学号、专业、指导教师等）
- 科学依据部分
- 研究内容部分
- 研究方法、技术路线、实验方案及可行性分析
- 研究计划及预期成果
- 指导教师意见及签名栏

## 🚀 快速开始

### 1. 安装 Tectonic

本项目使用 [Tectonic](https://tectonic-typesetting.github.io/) 进行编译。

```bash
# 使用 cargo 安装
cargo install tectonic

# 或使用包管理器
# macOS
brew install tectonic

# Windows (Scoop)
scoop install tectonic
```

### 2. 安装字体

模板使用以下字体：

**中文字体：**
- SimSun (宋体) - 正文
- SimHei (黑体) - 标题

**英文字体：**
- Times New Roman - 英文正文

### 3. 编译文档

```bash
# 使用 tectonic 编译
tectonic -X compile main.tex

# 或简写
tectonic main.tex
```

## ✏️ 使用方法

### 1. 填写基本信息

在 `main.tex` 中填写开题报告的基本信息：

```latex
% ------------------------------------------------------------
% 开题报告基本信息设置
% ------------------------------------------------------------
\topicname{你的论文题目}           % 题目
\studentname{张三}                 % 姓名
\studentid{2024123456}            % 学号
\major{计算机科学与技术}           % 专业
\advisor{李四}                     % 指导教师
\advisorid{20240001}              % 工号
\advisortitle{教授}               % 职称
\topicsource{教师科研}            % 课题来源
\projectname{智能排版系统研究}    % 依托项目名称
```

### 2. 填写各部分内容

```latex
% ------------------------------------------------------------
% 科学依据内容
% ------------------------------------------------------------
\scientificbasis{
    在这里填写科学依据内容，包括：
    - 课题的科学意义
    - 国内外研究概况、水平和发展趋势
    - 应用前景等
}

% ------------------------------------------------------------
% 研究内容
% ------------------------------------------------------------
\researchcontent{
    在这里填写研究内容...
}

% ------------------------------------------------------------
% 研究方法
% ------------------------------------------------------------
\researchmethod{
    在这里填写研究方法、技术路线、实验方案及可行性分析...
}

% ------------------------------------------------------------
% 研究计划
% ------------------------------------------------------------
\researchplan{
    在这里填写研究计划及预期成果...
}

% ------------------------------------------------------------
% 指导教师意见（留空，由导师手写）
% ------------------------------------------------------------
\advisoropinion{
    （此处由指导教师填写意见）
}
```

### 3. 生成开题报告

```latex
\begin{document}

% 生成开题报告书第一页
\makeproposal

% 生成开题报告书第二页
\makeproposalsecondpage

\end{document}
```

## 📝 各部分内容说明

### 科学依据
包括课题的科学意义、国内外研究概况、水平和发展趋势、应用前景等。

### 研究内容
详细描述本课题要研究的主要问题和具体内容。

### 研究方法
- **研究方法**：文献调研法、实验法、系统分析法等
- **技术路线**：研究的技术路径和步骤
- **实验方案**：具体的实验设计和实施方案
- **可行性分析**：技术可行性、时间可行性、资源可行性等

### 研究计划及预期成果
- 研究的时间安排和进度计划
- 预期取得的研究成果（论文、专利、软件等）

### 指导教师意见
由指导教师填写对开题报告的评价意见，并手写签名。

---

## ⚠️ 注意事项

1. **必须保留** `main.cls` 和 `main.tex` 两个核心文件
2. 确保系统安装了所需的 Windows 中文字体（宋体、黑体）
3. 使用 XeLaTeX 编译以支持中文
4. 各部分预留的空间是示例值，内容多时自动扩展
5. 指导教师意见栏留白，打印后由导师手写
